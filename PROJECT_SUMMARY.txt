  1. Open project

  3. Press Run (Shift+F10)

================================================================================
                              USAGE
================================================================================

1. Launch app → Selection Screen
2. Enter GitHub repo details (default: google/gson/main)
3. Press "Cargar Archivos"
4. Select .kt files with checkboxes
5. Press "Siguiente" → Review Screen
6. View code with syntax highlighting
7. Add comments in text field
8. Press "Guardar Comentario"
9. Navigate between files (Anterior/Siguiente)
10. Press "Resumen" → Summary Screen
11. View all comments
12. Press "Nueva Revisión" to start over
